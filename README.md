# 🤖📊 AI-Powered Sales Forecasting & Recommendation System

## 🌐 Live Demo

👉 https://sales-forcasting-system.vercel.app/login

---

## 📌 Overview

This project is an **AI-powered full-stack web application** that predicts future sales and generates intelligent product recommendations using machine learning models.

It combines a modern frontend with a Python-based backend to deliver data-driven insights for better decision-making.

---

## 🧠 Key Highlights

* 🤖 Machine Learning integrated into a real-world web app
* 🌐 Full-stack architecture (Frontend + Backend + Database)
* 📊 Data-driven prediction & recommendation system

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```id="u3v3aa"
git clone https://github.com/Sobika-E/Sales-Forecasting-System-With-Recommendation.git
cd Sales-Forecasting-System-With-Recommendation
```

### 2️⃣ Backend Setup

```id="rcbjv7"
pip install -r requirements.txt
python app.py
```

### 3️⃣ Frontend Setup

```id="fxgq3t"
npm install
npm start
```

---

## 🔄 Application Flow

1. User logs into the system
2. Inputs sales-related data
3. Flask backend processes request
4. ML model predicts future sales
5. Recommendation system suggests products
6. Results displayed on frontend
